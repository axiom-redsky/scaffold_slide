react-app-scaffold

바로 가져다 쓰는(scaffold)
React · TypeScript · Vite 기반

실무에서 곧바로 쓰는 UI 컴포넌트 · 유틸리티 · 예제 코드 · 가이드를 담은 스타터 프로젝트입니다. 프로젝트 초기 세팅과 반복 개발을 없애, 개발자가 비즈니스 로직에만 집중하게 합니다.

01

UI 컴포넌트

shadcn/ui를 래핑한 자체 컴포넌트를 @/shared/ui 한 곳에서 import.

02

유틸리티

전역 $util · $router · $ui 와 커스텀 훅으로 반복 로직을 바로 호출.

03

인터랙티브 데모

컴포넌트를 직접 조작해보며 동작과 옵션을 확인 — 학습 곡선을 낮춥니다.

04

코드 예제 및 가이드

복사해서 바로 쓰는 스니펫을 컴포넌트마다 함께 제공.

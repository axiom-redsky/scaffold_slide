React 기본 훅 위에,
데이터 통신을 useApi 하나로

TanStack Query를 감싼 단일 훅으로 조회·변경·로딩·에러·상태관리·캐시 무효화까지. 레이아웃 상태는 useSidebar로.

import { useApi } from '@axiom/hooks'

useApi()데이터 통신 · TanStack Query

GET목록 조회
useApi<User[]>('/users')
GET파라미터
useApi('/users', { params:{ id } })
POST검색 조회
useApi('/search', { method:'POST', type:'query' })
POST생성
const { mutate } = useApi('/users', { method:'POST' })
PUT수정
useApi('/users/1', { method:'PUT' })
DELETE삭제
useApi('/users/1', { method:'DELETE' })
CACHE캐시 무효화
invalidateQueries('/users')
RETURN반환값 · 자동
{ data, isLoading, mutate, isPending }
